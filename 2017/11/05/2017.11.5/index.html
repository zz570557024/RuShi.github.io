西塘

发表于 2017-11-05 | 分类于 Travel Notes

怀揣着满腹心事，周六去了西塘，江南六大古镇之一。

如果说白天的西塘是苏杭闺阁中的素颜，那么晚上的西塘就是江南青楼里的红装。当你为它日光下的婉约而心动时，它却在夜色里用妩媚诱惑着你。这里白日偶然还能寻得几处诗社文宅，经阁庙堂；夜里却尽是觥筹交错，音浪起伏。

西塘，像是每一个现代人的理想与欲望。平日里总是界限分明，但有时难免混为一谈。对西塘，这只是时间的问题；而于我，对理想保持憧憬，对欲望保持清醒。还没超然到“世人皆醉我独醒”的地步，但愿至少在我迷醉的时候，还能记起西塘的雨巷。

我想遇见你，即使在雨里。
